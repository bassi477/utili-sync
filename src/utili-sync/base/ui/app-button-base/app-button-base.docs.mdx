import {AppButtonBase} from './app-button-base';

## React Native Component to render a base for buttons.

The base provides a wrapper for the react-native component 'Pressable'. The props for pressable
are passed through making it completely configurable. The base can be used to create any style of
button in react-native.

### Component usage

```js
<AppButtonBase
  onPress={() => {
    /**
     * Perform action here.
     */
  }}>
  <Text>Button</Text>
</AppButtonBase>
```

### Using props to customize the text

Modify the text to see it change live:

```js live
<AppButtonBase
  onPress={() => {
    /**
     * Perform action here.
     */
  }}>
  /* Change child node to update button content */
  <Text>Button</Text>
</AppButtonBase>
```
